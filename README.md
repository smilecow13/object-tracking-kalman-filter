## Instruction

**Bước 1: Clone về**
```bash
git clone https://github.com/smilecow13/object-tracking-kalman-filter.git
cd object-tracking-kalman-filter
```

**Bước 2: Tạo branch riêng**
```bash
git checkout main
git pull origin main

# Tạo và chuyển sang nhánh riêng 
git checkout -b <tên-nhánh> (tuỳ)
```
**Bước 3: Push lên đúng branch**

```bash
git add .
git commit -m "vừa làm gì thì viết vào đây"
git push origin <tên-nhánh-của-bạn>
```

**Bước 4: Tạo PR (pull request)**

vào github

tìm và nhấn **Compare & pull request**


Create pull request và chờ review code và merge vào nhánh main.
